🧩 O que são Frameworks CSS?

Frameworks CSS são ferramentas que facilitam o desenvolvimento de interfaces modernas e responsivas. Eles oferecem classes, componentes e utilitários prontos para uso, economizando tempo e garantindo consistência visual.

🟦 Exemplo com Bootstrap

O Bootstrap usa uma abordagem baseada em componentes e grid system. Você pode montar layouts complexos apenas com classes HTML.

⚙️ Diferenças na Prática

Observe como o Bootstrap usa classes semânticas e o Tailwind classes mais descritivas (utilitárias).

Bootstrap:

<button class="btn btn-primary">Enviar</button>

Tailwind:

<button class="bg-blue-500 hover:bg-blue-600 text-white font-bold py-2 px-4 rounded">Enviar</button>
