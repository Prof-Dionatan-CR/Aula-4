CSS Grid Layout

Aprenda a organizar elementos em linhas e colunas com precisão e facilidade.

🔹 O que é o CSS Grid

O CSS Grid Layout é um sistema bidimensional que permite criar layouts complexos com linhas e colunas. Diferente do Flexbox, que organiza os elementos em um único eixo (horizontal ou vertical), o Grid trabalha com os dois eixos simultaneamente. 

.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
}
